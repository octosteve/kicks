# Kicks
A Sneakers Clone for Crystal lang

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     kicks:
       github: octosteve/kicks
   ```

2. Run `shards install`

## Usage

```crystal
require "kicks"
```
